Fazendo Layouts Responsivos

Layout Responsivo dentro do css

			img,video{
			max-width:100%; <- para a tamanho da tela do video e da imagem se ajuste com a tela
			}
			nav ul{
				height:auto; <- altura automatica
			}
			nav li{
				display:inline;
			}
			@media all and (min-width:960px){ /* apartir de 950px */
				body{
				width:960px;
				margin:auto;
				}
				}
			@media all and(max-width:480px){ /* apartir menos do que 480px abaixo de 480px */
				nav{
					height:100%;
				}
				nav li{
					display:block;
					padding:10px;
					margin:0px;
				}
				
			}
